## 第1章 · 开始二人之旅

### 1.1 什么是机器学习？

机器学习是让计算机从数据中自动学习规律、做出预测的技术。它不依赖人类编写固定规则，而是通过算法从大量的历史数据中发现模式。

举个简单的例子：如果你要在电商网站上投放广告，你需要知道投多少钱能带来多少点击量。没有机器学习时，你可能凭经验猜测；有了机器学习，你可以用历史数据训练出一个预测模型。

### 1.2 机器学习的三大类型

**回归（Regression）**：处理连续值的问题。  
- 问题示例：投入广告费 200 元，能获得多少点击量？
- 输出结果是一个连续的数值，比如 526 次点击。

**分类（Classification）**：处理离散值（类别）的问题。  
- 问题示例：这张图片是猫还是狗？
- 输出结果是有限的几个类别之一。

**聚类（Clustering）**：将数据自动分组，不需要预先知道类别。  
- 问题示例：把用户按照购买行为分成不同群体。
- 属于"无监督学习"，因为没有预先标注的正确答案。

### 1.3 监督学习 vs 无监督学习

- **监督学习**：训练数据（输入）带有正确答案（标签/标签值）。你告诉模型"对于这个输入，正确答案是这个"。回归和分类属于监督学习。
- **无监督学习**：训练数据没有标签。模型必须自己发现数据中的结构和模式。聚类属于无监督学习。

本书重点讲监督学习中的回归和分类，这是机器学习最基础和最重要的两个领域。

### 1.4 机器学习的通用流程

无论你处理什么问题，机器学习的基本流程都相似：
1. **收集数据**：获取带有标签（或没有标签）的训练数据。
2. **定义模型**：选择一个数学模型（如一次函数、多项式等），它包含一些未知参数。
3. **定义目标函数**：衡量模型预测结果与真实答案之间的差距（误差或似然）。
4. **优化参数**：通过数学方法（如梯度下降）不断调整参数，最小化（或最大化）目标函数。
5. **评估模型**：用测试数据检验模型在未见过的新数据上的表现。
6. **部署使用**：将训练好的模型应用到实际问题中。
